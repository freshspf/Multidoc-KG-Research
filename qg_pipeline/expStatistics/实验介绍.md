# 多智能体知识图谱实验报告

## 实验目标

本实验旨在构建一个基于LangGraph的多智能体知识图谱管道，用于从学术论文中自动提取知识图谱、评估质量并生成多跳推理问答对。具体目标包括：

1. **知识提取**：从学术论文的各个部分提取结构化知识图谱，以TTL/RDF格式存储
2. **质量评估**：使用GPT模型评估提取知识图谱的质量，确保达到预设标准
3. **问答生成**：基于高质量知识图谱生成多跳推理问答题，支持复杂的学术理解
4. **规模化处理**：支持大规模学术论文的批量处理，优化处理效率和资源利用

## 实验配置

### 技术架构

**多智能体系统**：
- **SectionBasedExtractor**：基于章节的知识提取智能体
- **TTLEvaluator**：TTL知识图谱质量评估智能体
- **MultiHopQAGenerator**：多跳问答生成智能体

**核心技术栈**：
- **LangGraph**：多智能体协调和工作流管理
- **OpenAI GPT模型**：智能体的核心推理引擎
- **TTL/RDF**：知识图谱存储格式
- **Python**：主要开发语言

### 配置参数

**API配置**：
```yaml
api:
  openai_api_key: ${OPENAI_API_KEY}
  openai_base_url: ${OPENAI_BASE_URL}
```

**模型配置**：
```yaml
extractor:
  model: "gpt-4o-mini"
  temperature: 0.1
  max_tokens: 3000

evaluator:
  model: "gpt-4o-mini"
  temperature: 0.1
  threshold: 6  # 评估通过阈值

qa_generator:
  model: "gpt-4o-mini"
  temperature: 0.7
  max_tokens: 800
```

**大规模处理配置**：
```yaml
large_scale:
  batch_size: 50              # 每批次论文数量
  max_concurrent: 4           # 最大并发批次数
  enable_parallel: true       # 启用并行处理
  checkpoint_interval: 10     # 进度保存间隔
```
